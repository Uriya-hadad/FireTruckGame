README

The purpose of the exercise:
to implement a program in the JAVA language by using a swing/awt package (GUI) by using different types of listeners (action, key) and using panels and the Graphics class.

The game idea:
You play a day in the life of Sammy the fireman, your goal is to put out as many fires as possible when each fire has a specific time where it should be put out!

Move with the truck by the arrow keys and splash the water by the space bar.

How to run the program:
The main function is in the game class.
